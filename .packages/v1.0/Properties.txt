Name
----
su.addToCalendar - Calendar Event Creator

Description
-----------
su.addToCalendar is a simple event creator plugin for calendar services. It supports iCalendar(.ics) format which supports most of mail applications, Gmail and Outlook Online. Made with jQuery.

You can use any HTML element with this plugin. It just needs attributes on element and initialization script on page.
